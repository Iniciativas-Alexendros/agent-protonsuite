# Code of Conduct

This project adopts the **Contributor Covenant 2.1**.

The full text is available at: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

Concerns about behaviour in this community can be reported privately to **conduct@alexendros.me**. Reports are reviewed by the maintainer; the reporter's identity is kept confidential. Outcomes range from a private warning to a permanent ban from the project's spaces (issues, discussions, PRs).

## Scope

This Code applies in all project spaces (GitHub repository, discussions, issues, PRs) and when an individual is officially representing the project.
